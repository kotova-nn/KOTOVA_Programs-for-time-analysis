ФУНКЦИИ
get_user_id() - запрашивает или получает из аргументов командной строки ID пользователя.

parse_date(date_str) - преобразует строку в объект datetime, пробуя разные форматы даты и времени.

merge_intervals(intervals) - объединяет пересекающиеся временные интервалы.

total_duration(intervals) - вычисляет суммарную длительность интервалов.

get_academic_year(date) - определяет учебный год по дате в формате YYYY-YYYY.

is_holiday(date) - проверяет, является ли дата праздничной (на основе RUSSIAN_HOLIDAYS).

is_weekend(date) - проверяет, является ли дата выходным днём (суббота или воскресенье).

get_school_days_in_year(academic_year_str, exclude_holidays_weekends) - рассчитывает количество учебных дней в учебном году.

safe_filter(df, user_col_candidates, df_name, user_id) - безопасно фильтрует DataFrame по user_id, пробуя разные названия колонок.

safe_get_columns(df, candidates, default=None) - безопасно находит нужную колонку в DataFrame по списку возможных названий.

filter_df_by_period(df, start_date, end_date, date_columns) - фильтрует DataFrame по временному периоду, оставляя записи в заданном диапазоне.

select_period() - предоставляет интерактивный выбор периода анализа (всё время / триместр / произвольный диапазон).

load_filtered_data(user_id) - загружает все CSV-файлы из DATA_PATH и фильтрует их по user_id.

compute_time_intervals(data) - собирает все временные интервалы активности из тренировок, ответов, медиа-сессий и действий.

analyze_time_and_days(data, intervals) - анализирует общее время на платформе, уникальные дни, распределение по учебным годам.

analyze_performance(data) - анализирует успеваемость: баллы, решённые задачи, тренинги, награды, сравнение с другими учениками.

analyze_course_time(data) - анализирует время просмотра видео в разрезе курсов.

analyze_weekly_activity(data) - анализирует распределение активности по дням недели, определяет самый активный день и тип активности.

get_hour_from_time(time_str) - извлекает час из строки времени.

classify_session_time(hour) - классифицирует час как утро (04-12), день (12-18) или вечер/ночь (18-04).

get_confidence_level(n) - определяет уровень уверенности (high/medium/low/insufficient) по количеству сессий.

analyze_circadian_rhythm(data) - анализирует биоритмы пользователя, определяет тип (жаворонок/сова/смешанный).

collect_productivity_data(data, start_date, end_date) - собирает данные о продуктивности по дням (видео, задачи, баллы, награды).

find_most_productive_day(data, start_date, end_date) - находит самый продуктивный день в заданном периоде.

analyze_productive_day(data, start_date, end_date) - выводит информацию о самом продуктивном дне с детализацией.

determine_archetype(data, user_id, weekly_info, rhythm_info) - определяет тотемное животное на основе метрик качества, активности, настойчивости и внимательности.

plot_results(user_id, data, time_info, perf_info, course_time, weekly_info, rhythm_info, start_date, end_date) - строит графики и диаграммы для визуализации результатов анализа.

main() - главная функция, управляющая всем процессом: загрузка данных, расчёт метрик, анализ, вывод результатов.
